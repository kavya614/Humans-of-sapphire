# Humans of Sapphire

A celebration of inspiring teachers and staff members at Sapphire Institution.

## Features

- **Browse by Month**: Select any month to view featured profiles
- **Browse by Year**: Filter profiles by specific years
- **Search Functionality**: Search for teachers by name
- **Profile Cards**: View detailed information about each person including their achievements

## How to Use

1. Click on any month button to see available profiles for that month
2. Select a year to view the profile for that month and year
3. Use the search box to find a teacher by name
4. View detailed achievement information on profile cards

## Getting Started

Simply open `index.html` in your web browser to run the website locally.

## Data

The website currently includes sample profile data. You can modify the `data` array in the JavaScript section of `index.html` to add more profiles.

Example format:
```javascript
{
  name: "Teacher Name",
  month: "January",
  year: "2026",
  photo: "https://image-url.com/photo.jpg",
  achievement: "Achievement description"
}
```
